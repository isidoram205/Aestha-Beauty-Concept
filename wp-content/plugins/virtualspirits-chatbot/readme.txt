VirtualSpirits Chatbot and LiveChat for your WordPress site

== Description ==

Create a Chatbot for your Website with VirtualSpirits. Chatbot Templates for Lead Generation, Customer Service, Ecommerce and Sales. Platform includes Chatbot, LiveChat, Web control panel, integrations and more. 30 Days Free Trial.

== Installation ==

1. Upload folder `/virtualspirits-chatbot/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your VirtualSpirits in Menu page
